## Homework 2: Vector Search

Reproducible solution for LLM Zoomcamp homework 2 using the local ONNX embedder.

### Files

- `download.py` downloads `Xenova/all-MiniLM-L6-v2`
- `embedder.py` loads the tokenizer and ONNX model
- `homework.py` computes answers for Q1-Q6 against commit `8c1834d`

### Run

Use the local virtualenv on Windows:

```powershell
.\.venv\Scripts\python.exe homework.py
```
